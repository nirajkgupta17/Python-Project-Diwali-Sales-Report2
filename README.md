# Diwali Sales Report – Data Visualization Project

# Project Overview

The Diwali Sales Analysis Report was developed using Python (Pandas, Matplotlib, and Seaborn) to analyze customer purchasing behavior during the festive season. The objective was to identify key sales trends, high-performing customer segments, and revenue-driving product categories.

The project involved data cleaning, exploratory data analysis (EDA), and visual storytelling through interactive and insightful visualizations.

# Tools & Technologies Used

Python

Pandas – Data cleaning and manipulation

Seaborn – Statistical data visualization

Matplotlib – Custom chart formatting and styling

Jupyter Notebook

# Key Analysis & Insights

1️⃣ Sales by Gender

Female customers contributed significantly higher revenue compared to male customers.

Indicates stronger festive purchasing behavior among women.

Visualization Used: Bar Plot (Seaborn)

2️⃣ Sales by Age Group

The 26–35 age group generated the highest sales.

Followed by the 36–45 segment.

Younger adults are the primary target demographic.

Visualization Used: Grouped Bar Chart

3️⃣ Sales by State

Top performing states:

Uttar Pradesh

Maharashtra

Karnataka

Indicates regional sales concentration.

Visualization Used: Horizontal Bar Chart (Sorted Descending)

4️⃣ Marital Status Analysis

Married women showed the highest purchasing power.

Indicates strong family-oriented festive shopping patterns.

Visualization Used: Countplot with Hue

5️⃣ Product Category Analysis

Highest selling categories:

Food

Clothing & Apparel

Electronics

Food category generated the maximum revenue.

Visualization Used: Sorted Bar Plot
